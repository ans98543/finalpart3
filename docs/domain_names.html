Domain Name

A domain name is a network address identifier. Not having domain names being generated would make network addressing harder. Imagine having to remember each IP address for each website, this would be very hard and time consuming.
The domain structure was created so people can have a better method to access websites. Thanks to domains multiple IP addresses are able to be identified. Also, Domains are used to track URLs you might know the popular ones such as, .com, .net, .gov, and etc. The internet core identifier is IP addresses so web pages have a DNS to allow the correct path to be accepted.

ICANN

The ICANN is known as the Internet Corporation for Assigned Names and Numbers. This non profit business manages and structures multiple databases. This company was founded in 1998 and there focused on managing IP addresses and DNS rooting.
More importantly , the ICANN has a big role in locating and establishing web pages. Thanks to ICANN it makes numerous companies' jobs easier by providing a stable operating system throughout the internet.

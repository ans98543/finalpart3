HTTP

This HTTP acronym stands for Hypertext Transfer Protocol. In web browsing, each weblink has a unique web address and this address communicates with internet servers. The servers ask for a website and to retrieve that data it is translated by the HTTP, which shows how communication is done. This is a known get request. For example, if you wanted to access Tumblr you will need the HTTP to establish this connection with the servers and this is done in less than a second.
In addition, HTTP contains an HTML document and this is the structure of the content displayed in the browser. If a website has a lot of images each image has its own HTTP request so images can be displayed. HTTP also sends info too, this happens when you fill out the info in a browser for submission then this info is translated in plain text and sent to servers. This is known as a post request and along with a cookie Id number for that website is established so you can be identified quicker when you visit the website again.

HTTPS

There is another form of HTTP and this is known as Hypertext Transfer Protocol Secure. This is to stop the tampering of people's info while web browsing. The secure socket layer and Transport layer are the security blocks for tampers. You will know if a website is secure if it has the lock logo in a web search bar. This makes sure that your HTTP requests are secure. Digital certificates are given to acknowledge that a secure connection is established.
